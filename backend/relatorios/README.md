# 📊 Relatórios Gerados

Esta pasta contém os relatórios gerados automaticamente pelo sistema.

## 📁 Estrutura

Os relatórios são gerados diariamente e incluem:

- **Relatórios de vendas** (PDF)
- **Relatórios de pedidos** (XLSX)
- **Relatórios de estatísticas** (CSV)

## 🔒 Segurança

**IMPORTANTE**: Esta pasta está no `.gitignore` para proteger dados sensíveis.

Os arquivos gerados aqui **NÃO** são versionados no Git e devem ser:
- Armazenados localmente apenas
- Enviados por email aos destinatários
- Removidos periodicamente para economizar espaço

## 🗑️ Limpeza

Os relatórios são mantidos por 30 dias e depois removidos automaticamente.

## 📝 Nota

Este README é o único arquivo desta pasta que é versionado no Git.
